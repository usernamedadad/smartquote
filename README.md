# SmartQuote 报价单编辑器

河南中科起重公司内部工具 —— 快速生成专业的工业起重设备 PDF 报价单。

## 快速开始

```bash
# 需要 Node.js 24 或更高版本 → https://nodejs.org
npm start
```

浏览器打开 **http://localhost:5173**，用管理员账号登录：

| 用户名 | 密码 |
|--------|------|
| `admin` | `admin123` |

管理员登录后可在「用户管理」中创建销售账号，分发给销售人员。每个销售只能看到和操作自己的报价项目和图片，互不干扰。

开发模式（文件修改自动重启）：

```bash
npm run dev
```

## 功能概览

### 📋 项目管理

- 项目列表以卡片形式展示，每张卡片顶部显示**真实的报价单缩略预览**
- 卡片网格首个位置为「+」新建按钮，点击即可创建报价单
- 顶部显示用户头像、名称和角色标签
- 支持按客户名称、报价编号、产品类型搜索
- 支持重命名、删除、复制项目
- 右侧信息栏显示最近编辑和系统统计

### 👤 个人中心

- 蓝色渐变 banner + 首字母头像 + 角色标签
- 统计卡片：项目数量、图片数量、登录账号
- 联系方式编辑：公司名称、联系人、WhatsApp、邮箱、网站、电话（编辑后自动应用到新建的报价单）
- 密码修改

### ✏️ 模块化编辑器

编辑器采用三栏布局：左侧模块导航 + 中间表单编辑 + 右侧实时预览，共 8 个编辑模块：

| 模块 | 说明 |
|------|------|
| **报价单标题** | 设置报价编号、日期、有效期、标题 |
| **公司信息** | 填写发件方公司名称、联系人、WhatsApp、邮箱 |
| **客户信息** | 填写收件方公司名称、联系人、WhatsApp、邮箱 |
| **产品参数** | 选择产品类型，配置规格参数，添加配件 |
| **产品图片** | 从图库选择图片，拖拽调整顺序（1-6 张自动适配网格布局） |
| **产品价格** | 设置数量、单价、行总价，自动汇总（支持运费项） |
| **条款和条件** | 添加/编辑条款（发货、付款、交期、包装等） |
| **页脚信息** | 设置报价单底部公司、网址、邮箱、电话 |

### 🏗️ 产品与配件

- 内置 **6 种起重设备** 参数模板：

  | 产品 | 说明 |
  |------|------|
  | 欧式葫芦 | FEM 2m 制，施耐德变频器及电气 |
  | CD 葫芦 | M3 制，手柄 + 遥控 |
  | 环链葫芦（运行式） | 带跑车，M3 制 |
  | 环链葫芦（固定式） | 固定安装，M3 制 |
  | 欧式单梁桥机 | A5 制，施耐德变频器 |
  | 桥机 + 支撑结构 | 桥机 + 钢结构支撑（含梁、柱、轨道、滑线） |

- 点击产品类型卡片即可添加/移除，支持上移下移调整顺序
- 每个产品可添加**配件**，配件拥有独立的名称、参数和价格
- 配件每行参数可自定义单位（如 pcs、set、kg），数量大于 1 时自动变复数
- 产品和配件以**分组容器**展示，蓝色左边框清晰标识归属关系

### 📸 图片管理

- 上传产品图片到共享图库
- 图片自动网格布局（1 张全宽 / 2 张两列 / 3 张上宽下两窄 / 4-6 张多列）
- 拖拽调整图片顺序

### 📄 PDF / HTML 导出

- 一键生成专业 PDF 报价单，自动命名下载
- 支持导出为独立 HTML 文件，无需服务器即可查看
- 导出前自动保存未保存的更改
- PDF 需要服务器安装 Playwright（不可用时优雅降级）

### ⌨️ 交互特性

| 操作 | 说明 |
|------|------|
| 实时预览 | 编辑内容即时反映到右侧报价单预览 |
| 拖拽排序 | 预览中的区块（双方信息、价格表、图片、条款、页脚）可拖拽调整顺序 |
| 撤销 / 重做 | 支持撤销最近 50 步编辑，或恢复到打开时的初始状态 |
| 批量导入 | 粘贴整段文本，自动识别字段并填入对应模块 |
| 缩放控制 | 预览支持适应面板、放大、缩小、全屏 |
| 未保存提醒 | 离开未保存的报价单时弹出确认对话框 |

### 🌐 多语言翻译

- 编辑好英文报价后，点击翻译按钮选择目标语言
- 自动翻译报价单文本和模板标签（如 "QUOTATION"、"Terms & Conditions" 等）
- 预览切换为翻译版本，确认后导出 PDF
- 支持阿拉伯语、希伯来语等 RTL 语言的自动排版
- 翻译由 Google Translate 免费接口提供，无需配置

## 局域网部署

让公司其他同事通过浏览器访问：

```bash
# 1. 启动服务
npm start

# 2. 查看本机局域网 IP
ipconfig

# 3. 放行防火墙（管理员终端，只需执行一次）
netsh advfirewall firewall add rule name="SmartQuote" dir=in action=allow protocol=TCP localport=5173
```

同事浏览器输入 `http://你的IP:5173` 即可使用。

## 技术栈

- **零外部依赖** — 仅使用 Node.js 24 内置模块
- **前端** — 原生 JavaScript ES Modules，无框架无构建
- **后端** — 手动路由 + 内置 SQLite（WAL 模式，启动/定时自动备份）
- **PDF** — Playwright 渲染（可选，不可用时优雅降级）
- **容错** — 项目数据打开时自动补齐缺失字段，全局异常捕获防止白屏
